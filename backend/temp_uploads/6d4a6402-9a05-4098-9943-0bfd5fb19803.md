# Privacy Policy

**Last Updated: January 15, 2024**

## Overview

LaTeX to WPS Converter ("the Extension") is committed to protecting your privacy. This privacy policy explains how the Extension handles your data.

## Data Collection

**The Extension does NOT collect any personal data.**

Specifically, the Extension does NOT:
- Collect any personally identifiable information
- Track your browsing history
- Send any data to external servers
- Use analytics or tracking tools
- Share any information with third parties

## Data Storage

The Extension stores the following data **locally on your device only**:

1. **User Preferences**
   - Theme selection (light/dark mode)
   - Last used LaTeX input (for convenience)

2. **Storage Method**
   - All data is stored using the browser's local storage API
   - Data never leaves your device
   - You can clear this data at any time by removing the Extension

## Permissions

The Extension requests the following permissions:

1. **contextMenus**
   - Purpose: To add "Convert LaTeX" option to the right-click menu
   - Usage: Only activated when you right-click on selected text

2. **storage**
   - Purpose: To save your preferences locally
   - Usage: Stores theme preference and last input for convenience

3. **activeTab**
   - Purpose: To access selected text on the current webpage
   - Usage: Only when you explicitly trigger conversion via menu or shortcut

## Third-Party Services

The Extension does NOT use any third-party services, analytics, or external APIs.

## Data Security

Since all operations are performed locally in your browser and no data is transmitted, there are no data security risks associated with external data transfer.

## Children's Privacy

The Extension does not knowingly collect any information from children under 13 years of age.
